# 前端界面层 (UID)

前端交互域的用户界面渲染与交互处理。

## 职责

- 显示服务与输入管理
- 应用沙箱运行时
- 系统界面层（状态栏、通知中心、多窗口）

## 支持设备

- **PC**：键盘/鼠标
- **移动**：触控/手写笔
- **其他**：统一抽象为逻辑输入事件

## 文件结构

```
frontend/ui/
├── README.md
├── display.c          # 显示服务
├── input.c            # 输入管理
├── compositor.c       # 窗口合成器
├── statusbar.c        # 状态栏
└── notifications.c    # 通知中心
```

## 沙箱运行时

所有原生第三方应用以 `.ssle` 格式部署：

```
/system/frontend/apps/
├── app1.ssle
├── app2.ssle
└── ...
```

每个应用启动时：
1. 分配独立 SELinux 标签
2. 创建文件系统命名空间
3. 默认无网络/存储/外设权限
4. 权限由用户授予，SED 动态下发
